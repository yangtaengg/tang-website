Tang! SMS Relay

Android 알림을
Mac 메뉴바로.

Tang!은 Android의 메시지/수신 전화 알림을 로컬 네트워크로 Mac에 전달합니다. 빠르게 확인하고, 필요한 경우 Mac에서 바로 액션을 이어갈 수 있습니다.

macOS 다운로드 Android 다운로드 릴리즈 보기

빠른 페어링

QR 또는 6자리 코드로 Android와 Mac을 간단히 연결합니다.

로컬 네트워크 우선

동일 Wi-Fi 환경에서 동작하며, 토큰 인증으로 기기 연결을 보호합니다.

실전 알림 흐름

메시지와 수신 전화 알림 중심으로 업무 흐름을 끊지 않고 확인합니다.

설치 가이드

  1. macOS ZIP을 내려받아 Tang!.app을 Applications로 이동
  2. Android APK 설치 후 알림 접근 권한 허용
  3. Mac 메뉴바 앱에서 Pair 화면 열기
  4. Android에서 QR 스캔 또는 코드 입력으로 연결

참고

macOS 앱은 서명되지 않은 빌드일 수 있어, 최초 실행 시 보안 안내가 표시될 수 있습니다. 최신 배포 파일은 GitHub Releases에서 확인하세요.
