# Tang Website

Tang! 홍보/다운로드를 위한 Next.js 14 기반 단일 랜딩 페이지입니다.

Public URL: https://yangtaengg.github.io/tang-website/

## Stack

- Next.js 14 (App Router) + TypeScript
- Tailwind CSS
- Framer Motion

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run start
```

## Link placeholders

링크 교체 위치는 `lib/constants.ts`입니다.
